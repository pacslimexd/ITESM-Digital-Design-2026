# ITESM-Digital-Design-2026
Central repository for the Digital Systems Design course

This repository contains all theoretical materials, installation guides, and the Verilog source code developed for the **Terasic DE10-Lite (Intel MAX 10 FPGA)** platform.

---

## 📂 Repository Structure

* **`/resources`**: Contains critical technical documentation and guides:
    * `DE10_Lite_User_Manual.pdf`: Hardware specifications and pinout.
    * `Syllabus - Digital Design Course.pdf`: Course timeline and objectives.
    * `Tutorial-DE10-Lite.pdf`: Basic board operation.
    * `QuartusI-MODELSIMI-BriefTutorial.pdf`: Step-by-step project creation guide.
* **`/WXX`**: Weekly course materials (Lecture Slides / PPTX).
* **`/WXX/exercises`**: Verilog source code (`.v`) and Project files.

---

## 🛠️ Required Software & Installation

To participate in the practical laboratory sessions, you must have the software installed correctly.

1.  **Software Link:** [Download Intel Quartus Prime Lite v25.1](https://www.intel.la/content/www/xl/es/software-kit/868561/intel-quartus-prime-lite-edition-design-software-version-25-1-for-windows.html)
2.  **Installation Guide:** Follow our [Step-by-Step Installation Guide](./resources/QuartusPrime/Quartus_Installation_Guide.md) to ensure you include MAX 10 support and the correct USB drivers.
